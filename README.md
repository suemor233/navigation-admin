## 采用Vue3+TSX编写的 navigation 后台管理系统

使用了以下技术

- Vue3
- Vite
- TSX
- Tailwindcss
- Pinia
- UmiRequest
- NaiveUI

# 开发

```sh
pnpm i
pnpm dev
```

# Reference
项目参考了 [mx-admin](https://github.com/mx-space/mx-admin)
